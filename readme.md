## For developers
`git clone https://github.com/ljosberinn/AuctionCraftSniper`

`cd AuctionCraftSniper`

`npm install`

### Compiling

 `npm run build` typescript conversion to source-mapped js

`npm run scss-dev-watch` scss conversion to source-mapped css
